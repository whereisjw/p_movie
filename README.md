# 무비매직(TMDB 영화사이트)

<aside>
💡 TMDB api를 사용한 영화사이트입니다.

</aside>

- **풀스택엔지니어 / 참여인원1명**
- **사용기술**
  - FE : Reactjs, Styled-components, Framer-motion, TMDB api
  - Infra: github

---

### ✅ 기능

- 영화 또는 TV쇼를 추천합니다.
- 모달창 형태로 상세페이지를 출력합니다.
- Swiper 라이브러리를 사용하지 않고 Framer-motion의 애니메이션을 이용해 슬라이드를 구현했습니다.
- Typescript를 사용해서 웹사이트를 구현하는 경험을 해보았습니다.
